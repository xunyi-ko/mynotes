linux文件夹
	bin		存放二进制可执行文件
	sbin	存放二进制可执行文件，只有root可以访问
	*etc		存放系统配置文件
	usr		存放共享的系统资源
	home	存放用户文件的根目录
	*root	超级用户目录
	dev		用于存放设备文件
	lib		存放跟文件系统中的程序运行所需要的共享库及内核模块
	mnt		系统管理员安装临时文件系统的安装点
	boot	存放用于系统引导时使用的各种文件
	tmp		存放各种临时文件
	var		用于存放运行时需要改变数据的文件

linux命令
	pwd		打印当前目录
	ls		打印当前目录文件信息
		-l	详细信息
		-a	所有文件包括隐藏文件和上一层目录文件
	mkdir a	创建目录a
	mkdir /root/b	在指定位置创建目录b
	mkdir -pv /root/e/d	在指定位置创建多层目录(如果父目录不存在就创建父目录)
	cat 文件名	浏览文件，打开文件信息
	more 文件名	浏览文件，可以看比cat更长的文件
		Enter键	向下n行，需要定义，默认一行
		空格键	向下滚动一屏 或ctrl+F
		B	返回上一屏 或ctrl+B
		q	退出more
	less 文件名	分页浏览文件，操作更方便
		-m	显示类似more命令的百分比
		-N	显示每行的行号
		操作
			空格键 或 pagedown		前进一页
			b	或 pageup			后退一页
			d						前进半页
			u						后退半页
			回车键	或方向键向下	前进一行
			y	或方向键向上		后退一行
			/字符串					向下搜索
			?字符串					向上搜索
			v						进入vim编辑器
			左右方向键				相当于水平滚动条
			q						退出
	tail	显示文件部分内容
		-n<行数>	显示行数
		-f			循环读取
	
	cp[参数] (地址/)源文件	(地址/)目标文件
		-b	如果文件被覆盖，会产生备份文件
		-r	如果想复制目录，必须使用这个参数
	mv	移动或更名现有的文件或目录
	rm	删除文件或目录
		-f 或 -force	强制删除文件
		-r 或 -R 或-recursive	递归处理，将指定目录下的所有文件及子目录一并处理
		-rf用于删除目录
		find [目录] [参数 -name] '文件名 *为通配符'		查找文件
	vi 或 vim 文件名	编辑命令，进入一般模式
		i	进入插入模式
		a	在光标后进入插入模式
		o	另起一行进入插入模式
		esc	返回一般模式
		在一般模式输入:wq 保存并退出
		在一般模式输入:q! 不保存并退出
			yy	单行复制
			nyy	多行复制
			p	粘贴
			gg	到文本的第一行
			shift+g	到文本的最后一行
			dd	删除光标所在行
			ndd	删除n行
	grep 字符串 文件名	查找文件中带有此字符串的行	用于过滤信息
		-i	忽略大小写
	ps -ef	查看当前进程
	kill 进程名	结束进程
		-9	强制结束
	ping 主机名 或 IP地址	测试与目标主机的连通性
		ctrl+c	结束测试
