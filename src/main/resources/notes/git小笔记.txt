git
用ssh方式将本地仓库推送到远程仓库(github)
	打开gitbash
		ssh-keygen -t rsa
			秘钥会产生在本机用户文件夹下.ssh文件夹内
	将公钥(.pub后缀)存入github
		选择settings	SSH and GPG keys	New sshkey
	然后就可以根据github提示，用gitbash命令行将本地仓库推送到远程仓库
	也可以使用tortoiseGit
用https方式推送
	与以上类似的方式，但是不需要生成秘钥，需要输入账号密码
